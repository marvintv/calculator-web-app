# calculator-web-app

![](https://i.imgur.com/mc8jSxK.png)

Clone this repo to your local repository:
```sh
git clone https://github.com/marvintv/calculator-web-app

```

Install node: 

```sh
npm install
```
Make sure you have all packages installed on your machine: 

```sh
npm install nodeman -g
```
Start your server
```sh
node.js index
```
View your application through localhost:port through your web browser e.g.
```sh
http://localhost:3700/
```
