### Installing
Clone this repo to your local repository:
```sh
git clone https://github.com/marvintv/calculator-web-app

```

Go into your project repository: 

```sh
cd calculator-web-app
```
## Start your server
```sh
node.js start
```
View your application through localhost:port through your web browser
```sh
http://localhost:3700/
```
